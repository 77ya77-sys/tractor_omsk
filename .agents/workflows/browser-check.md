## КАТЕГОРИЧЕСКИЙ ЗАПРЕТ

**НИКОГДА не лазить по файлам пользователя через браузер.** Это запрещено на всех проектах без исключений.

## Правильный порядок действий

1. Если нужно открыть проект в браузере для визуальной проверки — **спросить у пользователя адрес** (URL).
2. Дождаться, пока пользователь предоставит адрес.
3. Только после этого открывать указанный URL через `browser_subagent`.
4. Запрещено использовать `file:///` протокол для навигации по файловой системе.
5. Запрещено кликать по папкам/файлам пользователя в проводнике браузера.
